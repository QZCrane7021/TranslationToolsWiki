# 翻译工具百科

## 文档概要与阅读建议

本文档的作用是为有志于从事流媒体翻译工作的人提供技术信息，亦可为任何想要了解工作流中任意一种工具软件的人提供技术信息。

本文档同时会讨论翻译方面的观念、规范问题。

本网站将会讲解以下工具的基本功能：

- Clash Verge；
- yt-dlp；
- VideoCaptioner；
- Aegisub；
- Davinci Resolve。

如果你有翻译经验，但未从事过流媒体的翻译，请移步 “基本信息” 版块，了解过后，则再移步相关的功能版块；如果你没有翻译经验，请先移步 “翻译的原则” 版块进行了解。

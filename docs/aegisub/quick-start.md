# 快速上手

## 下载与安装

Aegisub 可以通过 [官网][official] 或 [Github Release][github-release] 下载。注意，官网中的 OS X 其实就是如今的 macOS。

[official]:https://aegisub.org/downloads/
[github-release]:https://github.com/TypesettingTools/Aegisub/releases

安装不多赘述。

## 用户界面

相比其他字幕打轴软件，Aegisub 将更多空间让位给了字幕编辑：

![Aegisub 的主面板](../assets/img/aegisub-panel.png)

///caption
Aegisub 的主面板
///

这难道说明 Aegisub 面对大量的字幕时间轴调整会很无力吗？答案并非如此。绝大部分字幕时间轴调整都是前后对齐，比如某句需要向后移动一段，和后一句重复了，后一句的开头也要后移和前一句结尾对齐。Aegisub 对此给出了一对右键菜单的功能，而这两个功能可以绑定快捷键，但默认为空，被雪藏了！

Aegisub 可以在打开字幕后再导入视频或音频，其中导入视频后时间轴计数可以由时间戳改为帧。

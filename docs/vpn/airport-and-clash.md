# 机场和 Clash

VPN 是一种抽象的服务，在现代通常由两个部分组成：机场和 Clash。

!!! warning "不要使用一体化的 VPN"
    有些时候你会看到一些 VPN 服务提供商的广告，宣称自家的 VPN 高速、稳定、方便，下载一体化软件即用，但实则风险很高。许多 VPN 服务提供商的负责人本人在国内，可能参与了多种灰黑产，很容易跑路或落网，因此不推荐选用。

## 机场

机场是指抽象的网络服务，在网上搜索即可找到。这些机场通常有按月计费的套餐，大多数价格相当便宜。付款后，它们往往会推荐你下载某些专门的软件接入服务，而这类软件就统称为 Clash。

为了机场和我自身的安全，我在此无法推荐某个特定的机场。请去互联网使用必应国际版搜索，或在各大群聊询问其他有经验者的意见，注意交流时要遵守 [VPN 简介][vpn-regulations] 中的守则。

[vpn-regulations]:./index.md#vpn-regulations

## Clash

Clash 的作用是和机场服务建立连接并正式 “翻墙”。VPN 工具本身是中立的，Clash 也是，因为它只是对特定的网址连接到远程服务，甚至可以设置为 REJECT，屏蔽广告。

目前，全平台支持面最广的 Clash 类软件有 Clash Verge、Clash Meta。本百科只讲解 Clash Verge，但其大体原理相通。

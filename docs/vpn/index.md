# VPN 简介

VPN，全称为 Virtual Private Network ，即**虚拟专用网络**。它通过软件和网络协议在公共网络（如互联网）上构建一条“ 虚拟网线”，实现不同网络或用户之间的安全连接。所谓 “虚拟”，意味着**不需要物理线路**；“专用” 则表示这条通道**只为特定用户或群体服务**，提供类似专用网络的安全和私密特性。

!!! danger "谨言慎行！"
    <span id="vpn-regulations"></span>
    VPN 在当前的语境下无异于所谓 “翻墙工具”。就算你借助它只浏览完全合法合规的内容，但总有人用它浏览色情、涉政等内容，使得它灰色化。因此，在使用 VPN 时，**千万要保证**：
    
    - 尽量**避免浏览违反国内法律的内容**，更不要发表触犯国内法律的言论，否则会带来**巨大的麻烦**，甚至可能牵连你使用的 VPN 服务商；
    - 不要在**任何具有官方性质的群聊**里提起相关事宜；
    - 在这方面有求于人时，注意不要直接使用 VPN 一词，转而使用：
        - 梯子 —— 泛指 VPN 服务；
        - 机场 —— 指 VPN 服务商；
        - Clash —— 泛指一类承接 VPN 服务的技术底层软件；
        - 魔法/科学上网 —— 泛指 VPN 服务。

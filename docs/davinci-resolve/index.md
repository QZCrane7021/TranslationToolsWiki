# Davinci Resolve 简介

Davinci Resolve 是由 Blackmagic Design 公司推出的综合视频编辑软件。

## Davinci Resolve 的优势

Davinci Resolve 的性能优化远强于剪映、必剪等基础剪辑软件，也好于 Adobe Premiere Pro。剪映、必剪没有整条的字幕轨道，想要调整字幕样式就必须全部选中，会导致严重卡顿甚至无响应崩溃。当然用户也可以分段选择，但操作会更复杂。

Davinci Resolve 的基础版功能依旧相当强大，对一般日用完全足够；付费版不像 Adobe Premiere Pro 那样采取订阅制，而是一次买断制，价格为 ￥2480，比 Adobe Premiere Pro 的一年订阅价还要便宜，对以视频创作为业的人来说并不算昂贵。

## Davinci Resolve 的劣势

Davinci Resolve 的各版块功能并不统一，而是由 Blackmagic Design 收购而来，因此 UI 较为割裂，导致用户难以掌握多版块的技能。

Davinci Resolve 设计之初偏向于影视而非流媒体，因此多轨道管理比较糟糕，超多轨道环境下性能反而不如 Adobe Premiere Pro，不利于 “美式鬼畜” 等需要大量拼贴素材的体裁。

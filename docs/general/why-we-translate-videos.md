# 为何翻译视频？

2017 年到 2019 年的哔哩哔哩，风头正盛。彼时，一群将 Youtube 视频搬运到哔哩哔哩的 UP 主引起了大家的注意。由于原生视频有时候会打上 Raw 的标签，而对于相当多的观众，看这种视频就像啃生肉一样难受，于是 “烤肉” 就成为了翻译视频的代称，“烤肉man” 就成为了这些视频译者的代称。

**你需要对烤肉这件事感兴趣，才适合成为烤肉man。**因为你大概率将要以转载渠道投稿，这样没有任何收入，需要兴趣支持。翻译视频以自制渠道投稿，虽然于情可接受，但于理不当。你获得的收益并不多，却容易受到别人的口诛笔伐。当你决定以自制渠道投稿，就意味着你必须接受因此带来的舆论风险，即使某些罪名是莫须有的。

!!! quote "哔哩哔哩社区公约"
    自制内容，须由UP主或其团队构思并创作完成、含有UP主或其团队的独创性表达。
    
    由本人创作、在原素材基础上进行二次创作加工或获得版权方授权的内容投稿时可选择自制；不满足自制要求的内容请统一选择转载。
    
    以下内容不属于自制范围:

    1. **搬运**：完整搬运他人作品或仅对他人作品做简单增删、视频内容表达与他人作品无实质差异，如仅添加拼接短暂片头、简单的片段前后顺序调换等；
    2. **无加工的纯片段截取**：加工指对原素材包括不仅限于添加特效包装、文字点评、剪辑制作、音频处理、解说配音等二次加工行为；
    3. **字幕**：未经版权方授权，自行添加的翻译字幕不属于自制类型；
    4. **录屏**：对投稿人没有参与制作、编辑、加工的作品，仅进行单纯录制或录播；
    5. 其他不属于自制的内容，包括不仅限于：
        1. 自行购碟压制上传、单纯倍速播放、倒放、镜像、调色、补帧等；
        2. 单纯对他人删除的创作内容进行补档。

无论你以转载还是自制渠道投稿，你都要在视频中融入自己的思考，在必要的时候查找资料，在长期的实践中确定你的风格。如此，你才能**和基于 AI 翻译的账号产生明显的区分**。

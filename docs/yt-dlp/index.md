# `yt-dlp` 简介

`yt-dlp` 是一款**免费开源的**命令行工具，用于下载 Youtube 上的视频，也兼容 Bilibili 的视频。这意味着工具没有 UI 界面，用户无法通过点击按钮、在输入框内输入内容就完成下载，而需要输入指令。但 `yt-dlp` 存在 UI 套壳软件，可以按需取用。本百科只讲解命令行工具的用法。

`yt-dlp` 需要真实用户的 Cookies 才能下载高清的视频，但这不是 `yt-dlp` 的缺陷，而是 Youtube 本身的限制。

!!! danger "千万不要共享 Cookies！"
    也许你可能动了和别人共享 Cookies 的想法，这样就不用注册账号了。但 Cookies 包含了用户的大量信息，一旦泄露，账号可能被盗、被封禁，因此千万不要共享！
